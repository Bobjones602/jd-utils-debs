# jd-utils-debs
Build Debian Package for JDownloader Appwork Utils

**Build dependencies**

- debhelper (>= 9)
- cdbs
- default-jdk
- maven-debian-helper (>= 1.5)
- junit4
- libcommons-lang-java (>= 2.6)
- libjackson-json-java
- libmiglayout-java
- libsvnkit-java (>= 1.8.6)
- [libproxy-vole-java](https://github.com/yadickson/proxy-vole-debs)

**Tested**

- Debian jessie
